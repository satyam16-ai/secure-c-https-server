Welcome

This blazing-fast, production-grade HTTPS server is written in C, using OpenSSL for secure connections. Serve static files, APIs, WebSockets, and CGI scripts—all with robust authentication and logging.

Features

  • 🔒 SSL/TLS via OpenSSL
  • ⚡ Ultra-fast static file serving
  • 🧩 REST API & WebSocket support
  • 🔑 HTTP Basic Authentication
  • 📝 CGI script execution
  • 📜 Configurable via server.conf
  • 📊 Access & error logging
  • 🧵 Multi-threaded for high concurrency

About This Project

Built from scratch in C for maximum performance and security. Designed for developers who want full control over their web server stack, with modern features and a clean, extensible codebase.
